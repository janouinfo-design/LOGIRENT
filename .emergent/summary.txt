<original_problem_statement>
The user wants to build a complete car rental solution.

**PRODUCT REQUIREMENTS:**

**1. Project Goal:**
   - Client Mobile App (iOS & Android)
   - Admin Web Back-office
   - (Optional future) Driver/Agent App

**2. Client App Features:**
   - **Account Management:** Email/password registration & login, email verification, password reset, user profile (Name, Phone, Address, Driving license photo upload).
   - **Vehicle Catalog:** View vehicles with photo, make/model, year, type, price/day, availability calendar, and options. Ability to filter by date, price, type, and location.
   - **Reservation:** Select start/end dates, choose a vehicle, add options, automatic price calculation, payment, and confirmation. Ability to modify and cancel reservations.
   - **Integrated Payment:**
     - Methods: Credit Card (Visa/Mastercard), Apple Pay/Google Pay, **TWINT (important for Switzerland)**.
     - Provider: Stripe recommended.
     - Features: Secure payment, automatic receipts, payment history, refunds.
   - **Notifications:** Push and email notifications for reservation confirmation, reminders, completion, cancellation, and payment status.

**3. Admin Back-office (Web):**
   - **Vehicle Management:** Add/edit/delete vehicles, manage status (available/rented/maintenance), photos, and pricing.
   - **Reservation Management:** View all reservations, change status, cancel, extend, and export to Excel/PDF.
   - **User Management:** View customer list, driving license documents, location history, and block accounts.
   - **Payment Management:** View transaction dashboard, export for accounting, process refunds, and generate automatic invoices.
   - **Statistics:** Dashboards for revenue, fleet utilization, popular vehicles, and revenue by period.

**4. Technical & Security:**
   - Encrypted data.
   - Hosting in Europe/Switzerland.
   - GDPR + Swiss LPD compliance.
   - Role-based access.
   - Activity logs.

**5. Recommended Architecture:**
   - **Frontend Mobile:** Flutter / React Native
   - **Backend API:** Node.js / Laravel
   - **Database:** **PostgreSQL**
   - **Hosting:** AWS Europe or Switzerland
   - **Payment:** Stripe + TWINT
   - **Image Storage:** S3 compatible
</original_problem_statement>
